ConfigMgrWebService

ConfigMgr WebService is a fork of a fork designed to extend the functionality of Operating System Deployment (OSD) within System Center Configuration Manager (SCCM) 2012 and above. This project provides additional methods that interact with both SCCM and Active Directory (AD), and resolves some issues with the parent fork relating to utilizing a specific domain controller for AD operations. This project is written in C#.

WindowsLAPSGUI

Windows LAPS GUI is a graphical user interface (GUI) that interacts with Active Directory (AD) to retrieve the password for a managed local administrative account on domain-joined computers. The managed account is determined by settings defined in the Local Administrator Password Solution (LAPS) group policy object (GPO). It is based on the SimpleLapsGui project. This project is written in C#.

jotm-mailing-list

The Jelly of the Month (jotm) mailing list allows interested individuals to sign up to receive a monthly newsletter along with a sample of that month's selected jelly. This project is written in HTML and CSS.
